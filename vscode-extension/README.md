## Usage

The extension starts automatically when you open VS Code.

- **Start/Stop**: Use `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac) and type `TouchFlow` to access commands.
- **Click the Status Bar**: Shows a summary of your session and options to reset.

## Configuration

You can customize the extension in VS Code Settings:

- `touchflow.idleTimeout`: Seconds of inactivity before pausing the timer (default: 5)
- `touchflow.showAccuracy`: Show/hide accuracy in status bar.
- `touchflow.showSessionTime`: Show/hide session timer.

## Install

1. Download the `.vsix` file.
2. In VS Code, go to Extensions (`Ctrl+Shift+X`).
3. Click the `...` menu at the top right.
4. Select "Install from VSIX...".
5. Choose the file.
